# Colory
> Fetches color name using hex value.


## Usage example


```python
Choose between xkcd or wiki color lists.
>>> a=Color('#000000','xkcd')
>>> a.name
'Black'

Mix Color
>>> a.mix('#FFFFFF')
>>> a
0x7f7f7f
>>> a.name
'Medium Grey'


```
